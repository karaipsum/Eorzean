## 字符预览
![字母表](preview/Glyphs.svg)
当前支持:
- 基础拉丁字母 (A-Z, a-z)
- 数字 (0-9)
- 常用标点（个人设计）
- 以及……？
## 字偶间距预览
![字偶间距](preview/Kerning.svg)
本字体以高质量曲线重绘了所有字符，并尝试统一了设计的对比度。
手工调节了字偶间距设定以提高可读性。可以在你喜欢的设计软件中选择“原始设定”以启用。
